## Run Locally

### 1. Clone repo

```
$ https://github.com/Gokulnathb19/E-Commerce_MEAN_App.git
$ cd E-Commerce_MEAN_App
```

### 2. MongoDB cloud & AWS S3

Use MongoDB Cloud & AWS S3 for database and file storage.

### 3. Run Backend

```
$ npm install
$ npm start
```

### 4. Run Frontend

```
# open new terminal
$ cd frontend
$ npm install
$ npm start
```
